변수 선언 및 값 할당하기

러스트 언어에서는 다양한 자료형을 지원하고 있지만 이번 강의에서는 단순한 예제를 위하여 정수형 변수를 선언하고 값을 할당해 보도록 하겠습니다. 변수 선언은 아래 예제1과 같이 let [변수 명]: [자료형] = [값]; 형식으로 합니다. 예제1의 2행 처럼 자료형을 명시하지 않아도 러스트 컴파일러에서 할당된 값을 보고 유추하기 때문에 컴파일 오류가 발생하지 않습니다. 이 때 주의할 점은 한 줄이 끝나면 반드시 세미콜론(;)을 붙여줘야 한다는 것 입니다.


let a: i32 = 5;
let b = 3;
